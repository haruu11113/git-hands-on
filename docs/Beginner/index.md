# GitHubとは
GitHubはgitのホスティングサービスです。  
Git用のwebサーバを提供しており、複数人で同じプロジェクトを進める場合によく用いられます。
Github以外にも下記に示すようなGitのホスティングサービスがあります。
- [GitHub](github.com)
- [GitLab](https://about.gitlab.com)
- [backlog](https://backlog.com/ja/)
- [bitbucket](https://bitbucket.org)
